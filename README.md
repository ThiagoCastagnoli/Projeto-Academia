# 📱 Sobre o Projeto

O **FitLife App** é um aplicativo Android desenvolvido em **Kotlin** voltado para treinos físicos e organização de exercícios.

O projeto conta com:
- 🔐 Autenticação via API
- 👤 Sistema de login e cadastro
- 📋 Listagem dinâmica utilizando RecyclerView
- 🔄 Navegação entre telas
- 🎨 Interface moderna e intuitiva

Desenvolvido com foco em:
- boas práticas de desenvolvimento Android
- arquitetura moderna
- organização de código
- evolução contínua como desenvolvedor mobile
